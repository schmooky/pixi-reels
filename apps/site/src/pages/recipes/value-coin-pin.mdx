Hold & Win-style coin collection, built on CellPin. Each coin carries its payout value in `payload.value`. The engine keeps them locked; game code reads the payloads to compute the total.

Previously the Hold & Win example used `CheatEngine._applyHeld()` — a test utility repurposed as production logic. Here, pins own everything: symbol identity, value, persistence.

## The whole mechanic

```ts
reelSet.events.on('spin:allLanded', ({ symbols }) => {
  for (let c = 0; c < symbols.length; c++) {
    for (let r = 0; r < symbols[c].length; r++) {
      if (symbols[c][r] === 'coin' && !reelSet.getPin(c, r)) {
        const value = rollCoinValue(); // server or client
        reelSet.pin(c, r, 'coin', {
          turns: 'permanent',
          payload: { value },
        });
      }
    }
  }
});
```

## Reading the total

Because `reelSet.pins` is a readable `ReadonlyMap`, computing the live total is trivial:

```ts
function currentTotal(): number {
  let total = 0;
  for (const pin of reelSet.pins.values()) {
    if (typeof pin.payload?.value === 'number') total += pin.payload.value;
  }
  return total;
}
```

The demo redraws this total on every `pin:placed` and `pin:expired`.

## Ending the feature

When the Hold & Win round ends, release everything:

```ts
for (const pin of [...reelSet.pins.values()]) {
  reelSet.unpin(pin.col, pin.row);
}
```

`pin:expired` fires with reason `'explicit'` for each one, giving you a hook to clear badges and animate the collect.
